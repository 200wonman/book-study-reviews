# 📌 소개

이 레포지토리는 내가 읽은 **프로덕트** 관련 책들의 리뷰와 학습 기록을 정리하는 공간입니다. 각 책을 읽으면서 **챕터별 내용 정리** 및 **핵심 파악**을 통해, 프로덕트 관리와 개발에 대한 이해를 깊이 있게 다집니다.

- **챕터별 내용 정리** 및 **핵심 파악**을 통해 책의 주요 아이디어와 핵심 메시지 이해
